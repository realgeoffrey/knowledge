### 基础
1. 三张表

    1. 资产负债表（Balance Sheet）

        资产负债表能够反映企业在特定日期的财务状况，其核心公式为：**资产 = 负债 + 所有者权益**。通过这张表，可以了解企业拥有哪些经济资源（资产），以及这些资源的来源，是通过债务融资（负债），还是股东投入（所有者权益）。分析资产负债表时，重点关注资产结构，即流动资产与固定资产的占比情况，以及负债率和净资产规模。

    2. 利润表（损益表，Income Statement / Profit and Loss Statement）

        利润表展示了企业在一定会计期间的经营成果，其核心逻辑为：**收入 - 成本 = 毛利**，**收入 - 成本 - 费用 = 净利润**。在实际分析中，**毛利率（毛利 / 收入）** 是衡量产品竞争力的重要指标，较高的毛利率通常说明产品具有较强的市场竞争力，能够在扣除直接成本后仍有较多的利润空间。**净利率（净利润 / 收入）** 则用于衡量企业的整体盈利能力，反映了企业在扣除所有成本和费用后的盈利水平。
    3. 现金流量表（Cash Flow Statement）

        现金流量表将企业的现金流量分为经营活动、投资活动和筹资活动三大板块。经营活动现金流反映了企业核心业务的现金收支情况，是企业现金的主要来源。自由现金流等于经营现金流减去资本支出，它是判断企业真实价值的关键指标之一。

    - 三张报表的数据关联关系

        1. 利润表中的净利润会影响资产负债表中的未分配利润
        2. 利润表中的净利润，作为现金流量表的现金流起点
        3. 现金流量表中的期末现金余额应该等于资产负债表中的货币资金（资产）
        - 利润表 = 这段时间赚没赚钱。现金流量表 = 钱到底有没有回来。资产负债表 = 到某一时点，公司还剩什么、欠什么。
        - 利润表解释“为什么资产负债表变化”。现金流量表解释“利润为什么没变成现金”。
1. 财务知识体系搭建

    1. 核心技能（0→1 阶段）

        1. 会计基础：学习**复式记账法**（Double-entry Bookkeeping），理解每一项经济业务都要以相等的金额，在两个或两个以上相互联系的账户中进行登记的原理，以及科目分类的规则，掌握各类会计科目的性质和用途，还有凭证录入的规范流程。可以参考《会计学原理》，这本书系统地介绍了会计的基本概念、原则和方法。
        2. 财务报表分析：深入研究三张报表之间的关系，即资产负债表、利润表和现金流量表之间数据的相互关联和验证关系。同时，掌握比率分析方法，通过《财务报表分析》等专业书籍学习流动比率、ROE（净资产收益率）等相关知识。
        3. 税务入门：了解增值税（VAT）、企业所得税（Corporate Income Tax）、个税（Personal Income Tax）等主要税种的计算规则，可参考国家税务总局官网的政策解读，获取最权威和最新的税收政策信息。

    待摊费用

    公司只要在一个会计期间的 期初和期末 各做一张资产负债表即可

    1. 应付票据：现金。应付账款：存货

        应付票据 因收到他人的借款而产生，获得现金；应付账款 因为赊账购买原材料等物品而产生，获得商品或服务。欠款需要未来某天偿还。

        应付账款的还款期限较短，通常是30天。应付票据的还款期限较长，可能会长达几年。这就是应付账款被列在负债类首位的原因，负债项目的排列通常根据各类债务到期的期限长短来列示。

        通常，应付账款不需要支付 利息，除非你无法按时还款。应付票据 通常需要支付利息。

    1. 支出

	    1. 价值大或意义长远的采购 支出使资产增加，从而被放入 资产负债表 中，即 支出 的 资本化，不会作为费用。
        2. 短期、少量、持续性的支出，也可作为 费用，称为 支出 的 费用化。
        - 对于一项支出，公司如何考虑其应该资本化还是费用化？取决于两个标准：

            1. 时间。该项支出使公司受益的时间。一般都会将受益时间超过一年的支出作为资本性支出，而若其受益期间仅在一年以内，就为费用性支出。
            2. 成本。如果你买了个垃圾桶，它的使用寿命超过了一年，你会资本化它吗？为什么不会？当然不会。因为它的价值太微不足道了。

                第二条标准是它的成本。大多数公司对此都会确定一个固定的金额，如500美元、1000美元、1500美元

    核销

    折旧（Depreciation）。直线折旧法（Straight-line Method）。账面净值（Book Value）。折旧是 费用（折旧费用），是 非付现费用（Non-cash Expense）。折旧 使 固定资产 减少。建筑物只能使用直线折旧法。设备可以选择直线折旧法或加速折旧法（如：双倍余额递减法，Double Declining Balance Method）。

    固定资产——设备，固定资产——建筑物。

    一般来说，资产根据其流动性的快慢依次列示于资产负债表中。（流动性是指资产变现的速度。）

    报表的左边是站在我们的角度上看到有什么东西流出了，我们的现金、存货等究竟是多少。报表的右边代表了账面上我们所拥有的东西，或者说左边的资产是由谁提供的。负债表明了我们对外的欠款，权益（包括留存收益）仅仅表明了我们对于这些资产占有的份额。

2. 利润表（损益表）

    利润表就如同一张动态的影片，因为它有开始和结束。这个时间段可以是一周、一个月、一个季度等。无论时间的长短，都称为会计期间（各财务报表的时间范围）。

    毛利 = 收入 - 成本

    毛利率 = 毛利 / 收入

    净利润 = 毛利 - 费用

    净利率 = 净利润 / 收入

    利润表将成本分为两类：商品销售成本，其囊括 了产品生产过程中所有的支出；费用，其囊括了企业经营中与产品生产 不直接相关的其他支出。对于那些没有有形产品的公司（即服务类行 业），这两类称为营业成本（又称服务成本）及费用。

    产品 生产的工艺是从 原材料（Raw Materials） 到 在产品（Work in Process，也称半成品） 再到 产成品（Finished Goods）。

    坏账（Bad Debt）：无法收回的应收账款，需要计提坏账准备。

    发给所有者的薪酬是一项费用
3. 现金流量表

    现金流量表仅仅记录现金 在给定期间内 的流入和流出。

    ![现金流量表](./images/6.jpg)

- 权责发生制（Accrual Basis），收付实现制（Cash Basis）。当有**存货**时，必须用权责发生制。可以用收付实现制来核算的公司：如服务行业，如诊所、律师行、会计师事务所、培训公司、咨询公司、房地产中介公司以及保洁公司等。

    >每年年末，当我们预测年终会盈利且要支付所得税款时，会想方设法合理地避税，于是我们会用现金购买大量存货，以加大销售成本，减少利润，达到少纳甚至不纳税的目的。但政府不会同意。因此有存货的企业只能用权责发生制进行核算

- 存货的计价方式：先进先出法（FIFO，First In First Out）、后进先出法（LIFO，Last In First Out）、加权平均法（Weighted Average）

    关于存货的计价采用先进先出法还是后进先出法对于企业来说非常重要，这主要取决于公司的两大经营策略：公司的纳税筹划策略是什么？公司所处行业的产品价格走势如何？​（当然，你无法预测未来几年产品的价格，但根据最近25个月的价格你可以预期到未来的价格趋势。​）

    >先进先出法和后进先出法仅仅是存货计价的方法（出于节税的目的），当真正耗用或出售存货时，不会按照这些会计方式进行，而是按照经营者判断的方式。

    一家公司若已经从先进先出法改为后进先出法核算，但要再一次改变，就不太可能获得美国国税局的批准。如果你已经从后进先出法转变到了先进先出法核算，需要补缴因此少计的所得税。

1. 会计基础

    1. 权责发生制和收付实现制

        核算制度，由于权责发生制相较于收付实现制可以更好地反映企业实际经营情况，我国要求企业必须使用权责发生制进行核算。
    1. 单式记账法和复式记账法

        记账方法。

        1. 单式记账法（Single-entry Bookkeeping）：我们日常所说的流水账，只记录收入支出的流水，而没有记录流水与账户的对应关系。
        2. 复式记账法（Double-entry Bookkeeping）：要求每一笔流水都记录在两个以上的账簿中，用于反映流水与账户的对应关系，并能够更清晰地看到每个账户的余额，我国企业所用的**借贷记账法**即是复式记账法的一种。
    1. 资产 = 负债 + 所有者权益

        所有者权益 又被称为净资产（Net Assets）
    1. 会计如何做账

        做账 其实就是把复式记账法 + 会计恒等式结合在一起，并写出会计的分录（Accounting Entry）。

        做账其实就是在反映资金在资产、负债和所有者权益三者中的流动关系，每一笔流水必须有2笔或以上对应的记录，用于反映资金流水与账户之间的关系

        - 借贷记账法（Debit-Credit Bookkeeping）：

            是一种以"账户"为核心、以借（Debit）和贷（Credit）作为记账方向、以会计恒等式始终平衡为约束的复式记账方法。

            核心原则只有一句：任何一笔经济业务，必须同时记录在两个或两个以上账户中，且借方金额 = 贷方金额。

            1. "借"与"贷"的含义

                在会计中，"借"（Debit, Dr.）和"贷"（Credit, Cr.）不再具有字面上的借入或贷出含义，它们仅仅是两个方向符号，类似于坐标轴的正负号。不可以写成+-。方向就是：debit或credit。值永远是正数。

                1. 借 (Left)：记在账户的左方。
                2. 贷 (Right)：记在账户的右方。

                至于具体哪一边代表增加，哪一边代表减少，取决于会计科目的性质：

                | 科目类别 | 借方（Debit, Dr.）Left | 贷方（Credit, Cr.）Right |
                | :--- | :--- | :--- |
                | 资产类（Assets） | 增加 | 减少 |
                | 成本/费用类（Expense） | 增加 | 减少 |
                | 负债类（Liabilities） | 减少 | 增加 |
                | 所有者权益类（Equity） | 减少 | 增加 |
                | 收入类（Revenue） | 减少 | 增加 |
    1. 会计分录

    >应收应付，这意味着货币是有时间价值的，直接的体现就是利息。如果你可以尽可能把现金掌控在自己手中，减少自己的应收款、增加自己的应付款，那么就有可能从利息中获益。
