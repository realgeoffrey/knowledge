### 语法
2. 换行 或 分号`;`表示语句结束
3. 注释

    1. `#`单行注释
    2. `""" """`或`''' '''`多行注释
4. 数据类型

    1. 整型`int`

        `0b100 == 4`、`0o100 == 64`、`0x100 == 256`
    2. 浮点型`float`

        `123.456 == 1.23456e2`
    3. 字符串型`str`

        单引号、双引号
    4. 布尔型`bool`

        `True`、`False`
5. 变量惯例：

    1. 惯例1：变量名通常使用小写英文字母，多个单词用下划线进行连接。
    2. 惯例2：受保护的变量用单个下划线开头。
    3. 惯例3：私有的变量用两个下划线开头。
6. `type(变量名或值)`检查类型
